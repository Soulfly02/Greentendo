### Global Controls
- **Power On/Off**: Click power button
- **Start**: Enter (pauses/resumes games)
- **Select**: Shift
- **A Button**: Space or A key
- **B Button**: B key or Escape
- **Menu Navigation**: Arrow Keys
- **Game Selection**: Enter or Space
